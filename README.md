# bilibili-grab

Download Bilibili videos and series at premium quality using yt-dlp + ffmpeg。

支持单个视频、多分P、合集/剧集下载。包含 B 站 412 错误、Chrome v127+ app-bound 加密等常见问题的解决方案。

## 安装

```bash
npx skills add Kedaarui/bilibili-grab
```

## 前置依赖

```bash
winget install yt-dlp.yt-dlp --accept-package-agreements --accept-source-agreements
winget install yt-dlp.FFmpeg --accept-package-agreements --accept-source-agreements
```
